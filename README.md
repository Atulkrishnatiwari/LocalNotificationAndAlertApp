## Overview

This Objective-C project showcases the implementation of Local
Notifications and Alerts in an iOS application. 
Local Notifications are used to schedule and display notifications within the app,
while Alerts are shown to the user with custom messages.

## Key Features

- **Local Notifications:** Schedule notifications to be displayed to the user, even when the app is in the background.
- **Alerts:** Display custom alerts with user-friendly messages.
- **Easy Integration:** Simple and straightforward integration into existing projects.

## Technology Used

- **Objective-C:** The primary programming language used for this project.
- **UIKit Framework:** Used for building the user interface and presenting alerts.
- **UserNotifications Framework:** Utilized for scheduling and handling local notifications.

### Prerequisites

- Xcode installed on your macOS machine.
- Basic knowledge of Objective-C and iOS development.
